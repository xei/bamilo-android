# 5 - Create the "BlackBerry ID Signing Token" and "Dev Certificate", 
# and save the file bbidtoken.csk on ~/.rim (Linux) or ~/Library/Research\ In\ Motion (Mac OS)
> https://developer.blackberry.com/codesigning/
> blackberry-keytool -genkeypair -keystore ACME.p12 -storepass rocket11 -author ACME

# or Eclipse (Windown->Preferences->BlackBerry Signing)
> Fill the "BlackBerry Id Token" and save
> Fill the "Developer Certificate" and save
> The BB Home should contain the "bbidtoken.csk" and the "author.p12".

# 6 - Configuration files on project
- assets/blackberry/certificate		(Token and certificate)
- assets/blackberry/android.cfg		(Push Notification Id)
- assets/blackberry/MANIFEST.MF		(Aplication definition: version, name, author)
- assets/blackberry/apk2bar.prefs	(Used for repacking)
- bb_repacking.sh					(Script for repacking and signed app)
